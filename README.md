# 🌷 Bhavithra Family Portal

A cute, simple family registration website.

Friends can open a link, fill a form, get an Application ID, and check their
approval status. Bhavithra (the admin) can review, approve/reject, and assign
roles from a password-protected page.

## What's inside

- `src/pages/Home.jsx` — landing page with 3 buttons
- `src/pages/JoinFamily.jsx` — registration form (with photo upload)
- `src/pages/CheckStatus.jsx` — friends check their own status by ID
- `src/pages/FamilyMembers.jsx` — public list of approved members
- `src/pages/FamilyTree.jsx` — visual tree centered on Bhavithra
- `src/pages/Admin.jsx` — password-protected admin control panel
- `supabase-setup.sql` — run once in Supabase to create the database
- `SETUP.md` — **start here** — full step-by-step setup & deploy guide

## Quick start

👉 Open **SETUP.md** and follow the 3 steps. No coding required — just
copy-pasting a few values. Takes about 15 minutes and gives you a real
shareable link.
